# 🛒 E-Commerce Fraud Detection

An AI/ML-based web application that detects fraudulent e-commerce transactions using Machine Learning models. This project helps identify suspicious transactions and reduce online fraud risks.

---

# 📌 Project Overview

E-Commerce platforms face different types of fraud such as:

* Transaction Fraud
* Fake Payments
* Review Fraud
* Refund Abuse

This project mainly focuses on **Transaction Fraud Detection** using Machine Learning techniques.

The system analyzes transaction details and predicts whether a transaction is:

* ✅ Legitimate
* ❌ Fraudulent

---

# 🚀 Features

* Fraud transaction prediction
* Machine Learning-based fraud detection
* Interactive Streamlit web application
* Data preprocessing and feature engineering
* Real-time prediction system
* Trained Random Forest model
* Fast and user-friendly interface

---

# 🧠 Machine Learning Models Used

* Random Forest ✅ (Final Selected Model)
* XGBoost
* Logistic Regression

> Random Forest was selected as the final model based on better recall performance for fraud detection.

# 📊 Model Evaluation Metrics

The following metrics were used to evaluate the models:

* Accuracy
* Precision
* Recall
* F1-Score
* Confusion Matrix

> Recall was considered the most important metric because detecting fraudulent transactions correctly is critical in fraud detection systems.
